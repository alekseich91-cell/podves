# Podves — калькулятор нагрузок на точки подвеса

Статический веб-калькулятор для быстрого расчёта распределения нагрузки по точкам подвеса грид-конструкций из прямых ферм. Для риггеров и техдиров шоу-ивент индустрии.

**Живая версия:** смотри раздел Pages этого репозитория.

## Что умеет

- Рисовать плоский грид из ферм в 2D-редакторе (клик-клик — ферма).
- Расставлять точки подвеса произвольно (на углах, мид-спан).
- Задавать погонный вес фермы и типы приборов с весами (штампом по ферме).
- Лебёдки в точках подвеса.
- Считать нагрузку на каждую точку методом рычага по пролётам.
- Показывать worst-case — что будет, если соседняя точка откажет.
- Подсвечивать перегруженные точки и фермы без связи с опорами.
- Автосохранение в браузере + экспорт/импорт `.json`.
- Undo/Redo.

## Как запустить локально

```bash
python3 -m http.server 8765
```

Открыть `http://localhost:8765/`.

Или просто открыть `index.html` в браузере (некоторые браузеры блокируют ES-модули из `file://` — тогда нужен локальный сервер).

## Тесты

```bash
npm test
```

## Стек

Vanilla JavaScript + SVG, без сборки и зависимостей. ES-модули загружаются браузером нативно, тесты — `node --test`.

## Ограничения (MVP)

- Только плоские гриды из прямых сегментов. Наклонные и 3D-конструкции не поддерживаются.
- Нет «ног» (ground support). Весь конструктив подвешен сверху.
- Расчёт упрощённый (рычаг + шарниры в точках) — для замкнутых контуров даёт чуть консервативнее точного FEM. Это инженерный инструмент для планирования, **не для сертификации**.
